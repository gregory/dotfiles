Repo for my dotfiles
============

This repo is aim to unify all my dotfiles

# Setup

```
git clone git@github.com:gregory/dotfiles.git && cd ./dotfiles && rake install
brew install ctags-exuberant
gem install zeus
```

Then Enter vim and do :BundleInstall
